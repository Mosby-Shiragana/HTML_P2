Count задаст кол-во столбов на которое поделится элемент
Fill задаст как заполнить столбы
Gap задаст зазор между столбами
Rule сокращенно задаст свойства color style width
Rule color задаст цвет между столбами
Rule style задаст стиль между столбами
Rule width задаст ширину между столбами
Span задаст сколько столбов должен занять элемент
Width задаст ширину столбов
Columns сокращенно задаст свойства width count
